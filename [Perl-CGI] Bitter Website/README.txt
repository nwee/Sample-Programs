Perl-CGI Bitter Website
-------------------

This project is a simple imitation of the Twitter service, called Bitter written in Perl-CGI. 

Features:
- Login Verification from a database
- Displaying a User profile and Bleats (Tweets) from a database
- Search functionality for users and Bleats.
- Creation, Tagging and Deletion of Bleats
- Listen(Follow)/Unlisten(Unfollow) features for users to other users.
